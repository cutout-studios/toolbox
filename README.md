# ✂️ The Cutout Toolbox 🧰

[![Maintainability](https://qlty.sh/badges/63ab5737-a9d3-4598-855e-83c7fe779ec6/maintainability.svg)](https://qlty.sh/gh/cutout-studios/projects/jsx)

The Deno-first stack we use to build Cutout's apps — JSX, web components, UI
primitives, and a native shell, all designed to stay close to their respective
platforms.

> [!CAUTION]
> These tools are either speculative or deeply in alpha: not for production use.
